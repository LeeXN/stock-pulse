隐私权政策 — Stock Pulse K线行情助手

最后更新 / Last updated：2026年6月12日

Stock Pulse（"我们"）非常重视您的隐私保护。本隐私权政策旨在说明 Stock Pulse 浏览器插件如何处理用户数据。请在使用前仔细阅读。

一、数据收集与处理（本地）

Stock Pulse 是一款本地优先的数据查看工具，不会主动上传、收集或传输任何个人身份信息。

以下数据仅存储在您的本地设备（Chrome 浏览器存储）中：

1.1 用户设置数据

  • 行情数据源偏好（如腾讯/东方财富/新浪等）
  • K线周期与显示数量（如日线/周线、显示120根K线）
  • 主题与界面配色
  • 刷新间隔（分钟）
  • 自选股列表（股票代码、名称、市场、自选分组）
  • 持仓数据（股票代码、买入价、持仓数量、备注）
  • 股票分组（自定义分组名称和成员）
  • 行情指数选择（如上证指数、深证成指等）

1.2 用户可选配置的第三方 API 密钥（仅在您主动填写时存储）

  • LLM（大模型）API Key：用于AI分析功能，支持 OpenAI / DeepSeek / 阿里通义 / Moonshot / OpenRouter / Google Gemini 等
  • LLM Base URL 与模型名称：用户自选的模型接入点配置
  • CORS 代理地址：用户自建代理配置，用于转发 LLM 请求
  • Tushare Pro Token：专业金融数据接口凭证
  • 聚合数据 Key：Juhe API 凭证

上述密钥由用户自愿提供，存储于浏览器本地存储中，我们不会获取、访问或传输这些密钥内容。

二、数据收集与处理（网络请求）

2.1 实时行情数据

为向您提供股票行情功能，插件会在您使用时向后述公开金融数据源发起请求（仅在您主动操作时触发，不含任何个人身份信息）：

  • 腾讯行情：qt.gtimg.cn（A股/港股实时报价）
  • 东方财富：push2.eastmoney.com、push2his.eastmoney.com、searchapi.eastmoney.com、datacenter.eastmoney.com、np-anotice-stock.eastmoney.com（行情、K线、财务、公告数据）
  • 新浪财经：hq.sinajs.cn、quotes.sina.cn、stock.finance.sina.com.cn（行情数据）
  • 聚合数据：web.juhe.cn（附加数据服务）
  • Tushare Pro：api.tushare.pro（专业金融数据，需用户自备Token）

2.2 AI 分析功能（LLM 调用）

当您使用 AI 分析功能（如财报解读、技术面分析、风险评估）时：

  • 插件会将您的投资相关问题（不含账户信息）发送至您配置的 LLM 提供商
  • 支持通过用户自备的 CORS 代理转发请求
  • 支持的 LLM 提供商：OpenAI、DeepSeek、阿里云通义千问、Moonshot AI（月之暗面）、OpenRouter、Google Gemini
  • 您的请求内容仅用于生成回复，不用于模型训练

三、权限使用说明

以下是插件申请的 Chrome 扩展权限及其用途：

  • storage：存储用户自选股、持仓、设置等本地数据
  • alarms：定时任务（定时刷新行情角标、保活服务）
  • sidePanel：在浏览器侧边栏展示行情面板
  • contextMenus：右键菜单功能（选中股票代码时显示"加入Stock Pulse自选股"和"在东方财富查看"）
  • host_permissions：访问上述金融数据源和 LLM API 端点的权限，仅在您主动操作时触发

四、数据共享

我们不会主动向任何第三方共享、出售或传输您的个人数据或设置数据。您在本插件中配置的 API 密钥仅存储于本地，不会被发送至我们运营者控制的任何服务器。

插件中指向外部金融数据源和 LLM 提供商的所有请求，均为您与相应服务提供商之间的直接交互，受各提供商的隐私政策约束。

五、数据安全

所有本地存储数据受 Chrome 浏览器的安全机制保护。我们建议您：

  • 不要在不安全的设备上保存敏感的 API 密钥
  • 定期清理不再使用的插件数据

六、用户权利

您可以随时通过以下方式管理或删除您的数据：

  • 导出数据：插件内置"导出数据"功能，可将所有本地数据导出为 JSON 文件
  • 删除数据：在插件设置中清除数据，或通过 Chrome 扩展管理页面卸载插件（会清除该插件的所有本地存储）
  • 重置设置：可在设置中恢复默认配置

七、未成年人

本插件面向成年投资者设计。我们无意收集未成年人信息。若您认为我们已收集了未成年人的信息，请联系我们予以删除。

八、政策更新

如我们对本政策进行重大变更，将通过插件更新或本页公告形式通知您。变更后继续使用插件即表示您接受更新后的政策。

九、联系我们

如您对本隐私权政策有任何疑问，请通过 GitHub Issues 或插件商店评论联系我们。

Privacy Policy — Stock Pulse K-Line Market Assistant

Last updated: June 12, 2026

Stock Pulse ("we", "us", or "our") is committed to protecting your privacy. This Privacy Policy explains how the Stock Pulse browser extension handles user data. Please read it carefully before using the extension.

1. Data Collection and Processing (Local)

Stock Pulse is a local-first market data viewer. We do not proactively collect, upload, or transmit any personally identifiable information (PII).

The following data is stored exclusively on your local device (Chrome browser storage):

1.1 User Settings

  • Market data source preferences (Tencent / East Money / Sina, etc.)
  • K-line period and display count (daily / weekly, up to 120 candles)
  • Theme and UI color scheme
  • Refresh interval (in minutes)
  • Watchlist (stock code, name, market, custom groups)
  • Portfolio holdings (stock code, purchase price, quantity, notes)
  • Stock groups (custom group names and members)
  • Market index selection (e.g., SSE Composite, SZSE Component)

1.2 Third-Party API Keys (only stored when you voluntarily provide them)

  • LLM (Large Language Model) API Key: for AI analysis features, supporting OpenAI / DeepSeek / Alibaba Tongyi / Moonshot / OpenRouter / Google Gemini
  • LLM Base URL and model name: your chosen model endpoint configuration
  • CORS proxy address: your self-hosted proxy for forwarding LLM requests
  • Tushare Pro Token: professional financial data API credential
  • Juhe API Key: Juhe data service credential

These keys are provided voluntarily by you and stored in the browser's local storage. We do not have access to, or control over, these credentials.

2. Data Collection and Processing (Network Requests)

2.1 Real-Time Market Data

To provide stock market data, the extension makes requests to the following public financial data sources when you actively use the relevant features (no personal information is included):

  • Tencent Finance: qt.gtimg.cn (A-share / HK-stock real-time quotes)
  • East Money: push2.eastmoney.com, push2his.eastmoney.com, searchapi.eastmoney.com, datacenter.eastmoney.com, np-anotice-stock.eastmoney.com (quotes, K-lines, financials, announcements)
  • Sina Finance: hq.sinajs.cn, quotes.sina.cn, stock.finance.sina.com.cn (market data)
  • Juhe Data: web.juhe.cn (supplementary data services)
  • Tushare Pro: api.tushare.pro (professional financial data, requires user-provided token)

2.2 AI Analysis Features (LLM Calls)

When you use AI analysis features (such as earnings report interpretation, technical analysis, or risk assessment):

  • The extension sends your investment-related questions (without any account information) to the LLM provider you configure
  • Requests may be forwarded through a user-provided CORS proxy
  • Supported LLM providers: OpenAI, DeepSeek, Alibaba Tongyi Qianwen, Moonshot AI, OpenRouter, Google Gemini
  • Your request content is used solely for generating responses and is not used for model training

3. Permission Usage

The following Chrome extension permissions are requested and their purposes:

  • storage: storing watchlist, portfolio, and settings locally
  • alarms: scheduled tasks (periodic badge refresh, service keep-alive)
  • sidePanel: displaying the market panel in the browser's side panel
  • contextMenus: right-click menu items ("Add to Stock Pulse Watchlist" and "View on East Money" when selecting stock codes)
  • host_permissions: permission to access the financial data sources and LLM API endpoints listed above — requests are only made when you actively use features that require them

4. Data Sharing

We do not share, sell, or transfer your personal data or settings to any third party. API keys configured within the extension are stored locally and are never sent to any server controlled by the extension developers.

All requests to external financial data sources and LLM providers are direct interactions between you and the respective service providers, governed by their own privacy policies.

5. Data Security

All local storage is protected by Chrome's security mechanisms. We recommend that you:

  • Avoid saving sensitive API keys on unsecured devices
  • Regularly clean up data from extensions you no longer use

6. Your Rights

You can manage or delete your data at any time:

  • Export data: the extension has a built-in "Export Data" feature to export all local data as a JSON file
  • Delete data: clear data in the extension settings, or uninstall the extension via Chrome's extension management page (this removes all local storage for the extension)
  • Reset settings: restore default configuration in settings

7. Children

This extension is designed for adult investors. We do not knowingly collect information from minors. If you believe we have collected information from a minor, please contact us for removal.

8. Policy Changes

If we make material changes to this policy, we will notify you through extension updates or by posting a notice on this page. Your continued use of the extension after changes constitutes acceptance of the updated policy.

9. Contact Us

If you have any questions about this Privacy Policy, please contact us via GitHub Issues or the Chrome Web Store review section.
